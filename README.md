# 萨莉亚

萨莉亚的点单页面很丑，所以自己写了个玩

## 技术栈

- Vite
- React
- TailwindCSS

## 开发&编译

```sh
# 开发
pnpm install
pnpm dev
```

```sh
# 编译
pnpm build
```
